# Assembler
Assembler for a Simple Instruction Set written in C

The Instruction Set is as follows :

OPCODE     IMPERATIVE-STATEMENT
0          STOP
1          ADD
2          SUBTRACT
3          MULT
4          MOVER
5          MOVEM
6          COMP
7          BC
8          DIV
9          READ
10         PRINT


The Assembler Directives to be used are :
ASSEMBLER DIRECTIVE

START
LTORG
EQU
ORIGIN
END
